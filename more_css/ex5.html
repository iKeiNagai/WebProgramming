Exercise 5

Maintaing Quality Control

  • Implement a standard external stylesheet all teachers must use
  • Discourage inline styles to ensure uniformity and easier maintanance
  • Establish design rules for heading, fonts, colors, and layouts
  • Limit permission to edit certain tags

Process to mantain consistent styling

  • Define a global external css file
  • Use internal CSS for page-specific adjustments
  • Allow only content update by teachers
  • Use consistent template system

The approach is better because it prevents inconsistent design across pages, reduces errors caused by CSS modifiers, and improves accessibility and readability for the page users.

HTML tags and CSS styles

    HTML TAGS
  • <div> for section containers
  • <h1> - <h6> for section headings
  • <table> for schedules and timetables
  • <li> or <ol> for subjects and assignmets lists
  • <a> for online resources
  • <img> for school logos or diagrams
    CSS styles
  • body and h1 for global styling
  • font-size and color for inline css
  • text- align for specific page adjustments
